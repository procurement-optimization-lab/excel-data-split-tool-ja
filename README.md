# Excelデータ自動仕分けツール（業務効率化）

膨大な行数のExcelデータを数秒で自動分割。  
手作業の繰り返しから解放されます。

---

## ■ 機能

- 任意の列（仕入先・品目コードなど）で分割
- 条件ごとに自動でファイル生成
- 書式コピーON/OFF
- ワンクリック実行
- プログラミング不要

---

## ■ Before / After

手作業：  
フィルター → コピー → 保存 → 繰り返し（数時間）

本ツール：  
「実行」クリック → 完了（数分）

- 他のExcelは閉じる
