# Excelデータ自動仕分けツール（業務効率化）

膨大な行数のExcelデータを数秒で自動分割。  
手作業の繰り返しから解放されます。

---

## ■ 機能

- 任意の列（仕入先・品目コードなど）で分割
- 条件ごとに自動でファイル生成
- 書式コピーON/OFF
- ワンクリック実行
- プログラミング不要

---

## ■ Before / After

手作業：  
フィルター → コピー → 保存 → 繰り返し（数時間）

本ツール：  
「実行」クリック → 完了（数分）

## ■ 使い方

1. 対象ファイルを同一フォルダに配置  
2. 条件入力  
3. 実行  

---

## ■ 注意事項

- Excelファイルは1つのみ配置  
- 他のExcelは閉じる  

---

## ■ 出力

- タイムスタンプフォルダに保存  
- ファイル名＝分割キー  
- 空白は「EMPTY」
